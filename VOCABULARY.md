# Vocabulary

This document gives each important term a single canonical definition.

## I

I is a living scientific infrastructure. It is a constitutional scientific infrastructure. It is a distributed scientific instrument. It is a public scientific infrastructure. It is a protocol for building living scientific infrastructure.

## The Repository

The repository is not the infrastructure. It hosts the current public definition of I. It is the Public Constitutional Archive of I.

## Instrument

An Instrument is a protocol for making attention durable. It defines what counts as a record within a bounded method.

## Host

A Host is a stewarded place of continuity. It carries part of I without owning it.

## Observation

An Observation is a record of attention. It is immutable.

## Raw Data

Raw Data are the preserved record from which knowledge is later derived.

## Analysis

An Analysis is a versioned interpretation of observations.

## Knowledge

Knowledge is derived, never primary. It is temporary and revisable.

## Stewardship

Stewardship is the duty of keeping I in order.

## Governance

Governance is the carrying of principle through protocol.
