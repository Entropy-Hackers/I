# Manifest

This repository is not the infrastructure. It is the Public Constitutional Archive of I.

I is not a project, an initiative, a platform, an organization, a startup, a framework, or a solution. It is a living scientific infrastructure. It is a constitutional scientific infrastructure. It is a distributed scientific instrument. It is a public scientific infrastructure. It is a protocol for building living scientific infrastructure.

A scientific infrastructure is not completed by ceremony. It is maintained by discipline. The record must remain intact. The interpretation must remain distinct from the record. The inheritance of form must not be confused with the authority to govern.

I is not being launched. I is being defined.

It is defined in language, in protocol, in stewardship, and in continuity. It is defined so that it may endure across generations of inquiry.
